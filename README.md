# Cursed Compass #

## About ##
The Cursed Compass is a compass that can show up in chest loot that normally may contain a Compass. The Cursed Compass is a cursed version of the vanilla Compass. Instead of leading you towards your spawn point it'll lead you towards danger. It will target a random monster and follow it's position.
